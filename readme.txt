README File:
This is our version of java terminal chess. This is a two player pass-and-play turn based game so make sure you
have a friend to play with you!  This game is played exactly like the original which has rules that can be found here: 
https://en.wikipedia.org/wiki/Chess. 
In order to play this game, in terminal, simply run:
$javac *.java 
followed by
$java Chess
Have fun playing!
